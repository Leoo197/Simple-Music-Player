# Simple Music Player

A lightweight, Python-based desktop music player. This script provides a simple and straightforward way to play local audio files directly from your machine.

## ⚠️ Important: Adding Your Own Music
To keep this repository lightweight and respect copyright, no audio files are included in this project. **You must provide your own music to use the player.**

1. Create a new folder named `Songs` inside the project directory (right next to `main.py`).
2. Add your favorite `.mp3` or `.wav` files into that `Songs` folder.
